Build Project

```
cd algorithms/java/
mvn clean package
```

Run examples

```
# with out inputs
java -cp target/algs-1.0-SNAPSHOT.jar com.linbo.algs.examples.convexhull.GrahamScan

# with inputs
java -cp target/algs-1.0-SNAPSHOT.jar com.linbo.algs.examples.collinear.FastCollinearPoints input6.txt
```
